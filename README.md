Doodle test project
===================

INSTALLATION
------------

```bash
$ npm install
```

REQUIREMENTS
------------

node.js

QUICK START
-----------

to build and start server

```bash
$ webpack && node ./build/server.js
```

to build frontend

```bash
$ cd frontend/
$ webpack
```

to run tests

```bash
$ npm run test
```
